# Types of Functions
Based on the degree of the function.

### Examples
* `f(n) = 5`
* `f(n) = 2n + 3`

| Complexity | Name |
| - | - |
| `O(1)`| Constant |
| `O(n)`| Linear |
| `O(log_2(n))` | Logarithmic |
| `O(n^2)` | Quadratic |
| `O(n^3)` | Cubic |
| `O(n^2)` | Quadratic |
| `O(2^n)` | Exponential |
